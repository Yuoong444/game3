# 체인 히어로즈: 퍼즐 RPG MVP

업로드된 기능명세서 기반의 정적 HTML/CSS/JS 모바일·웹 퍼즐 RPG MVP입니다.

## 실행 방법
1. 압축을 풉니다.
2. `index.html`을 브라우저에서 엽니다.
3. GitHub Pages에는 `index.html`, `style.css`, `game.js`를 그대로 업로드하면 됩니다.

## 구현된 핵심
- 6x6 4방향 연결 퍼즐
- 같은 색 3개 이상 연결 시 공격
- 50개 스테이지 자동 생성 데이터
- 제한 시간, 클리어/실패 판정
- 직선 폭발, 범위 폭발, 색 제거, 체인 강화 타일
- 캐릭터 30종 데이터, 획득/편성/레벨업
- 일일 미션, 업적, 간이 리더보드 대체 패널
- 모바일 우선 반응형 UI, 터치 드래그 지원
- localStorage 저장, 오프라인 안내

## MVP 한계
- 실제 소셜 로그인/서버 동기화/전역 리더보드는 백엔드가 필요하므로 구현하지 않았습니다.
- 사운드는 브라우저 자동재생 정책을 고려해 제외했습니다.
